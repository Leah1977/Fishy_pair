How to Play

The object of the game is to find all the pairs before the time runs out!

  1. Choose a level to begin.
  2. Click on any card to start the game.
  3. Click on another card to find a pair.
  4. Use your memory to find all the pairs before the time runs out.
  5. You have 60 seconds to find all the pairs.
  6. If you would like to restart the game click on the restart button.

Enjoy the game!
